# ❌⭕ XO AI Agent

---

An intelligent Tic-Tac-Toe AI agent designed to play optimally against human players using classic Artificial Intelligence techniques.

---

## 🚀 Live Demo

🔗 [Play the Game](https://xo-game10.netlify.app/)  

---


## 🎯 Problem Statement

Tic-Tac-Toe is a simple game, but creating an AI that plays optimally requires implementing decision-making strategies that simulate human thinking and ensure the best possible outcome.

---

## 💡 Solution

This project implements an AI agent capable of:

- Making optimal moves  
- Predicting opponent behavior  
- Avoiding losing scenarios  
- Ensuring either a win or a draw  

---

## 🧠 AI Approach

The AI agent is built using:

- Minimax Algorithm  
- Game Tree Search  
- Recursive decision-making  

The agent evaluates all possible moves and selects the best one based on maximizing its chances of winning while minimizing the opponent's chances.
